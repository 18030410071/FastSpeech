## Train
1. Put [LJSpeech dataset](https://keithito.com/LJ-Speech-Dataset/) in `data`.
2. Run `preprocess.py`
3. Run `train.py`

## Dependencies
- python 3.6
- pytorch 1.1.0
- numpy 1.16.2
- scipy 1.2.1
- librosa 0.6.3
- inflect 2.1.0
- matplotlib 2.2.2
